### Что такое тайлы?[](https://docs.2gis.com/maps/overview#%D1%87%D1%82%D0%BE-%D1%82%D0%B0%D0%BA%D0%BE%D0%B5-%D1%82%D0%B0%D0%B9%D0%BB%D1%8B)

Тайл — это квадратное изображение, соответствующее участку карты. Использование тайлов позволяет экономить ресурсы устройств и отображать только видимую часть карты, подгружая необходимые фрагменты при перемещении по карте или при изменении масштаба карты.

### Чем растровые тайлы отличаются от векторных?[](https://docs.2gis.com/maps/overview#%D1%87%D0%B5%D0%BC-%D1%80%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B2%D1%8B%D0%B5-%D1%82%D0%B0%D0%B9%D0%BB%D1%8B-%D0%BE%D1%82%D0%BB%D0%B8%D1%87%D0%B0%D1%8E%D1%82%D1%81%D1%8F-%D0%BE%D1%82-%D0%B2%D0%B5%D0%BA%D1%82%D0%BE%D1%80%D0%BD%D1%8B%D1%85)

Растровые тайлы — это изображения фрагментов карты в формате PNG, которые отрисовываются на стороне сервера и отправляются вашему устройству в готовом виде. Отображение растровой карты требует меньше ресурсов устройства, но не позволяет гибко настраивать внешний вид карты и её поведение. Чтобы работать с растровой картой, используйте [Raster Tiles API](https://docs.2gis.com/maps/others/rastertiles/overview) или [Static API](https://docs.2gis.com/maps/others/static/overview).

Векторные тайлы — это данные, которые ваше устройство получает и отрисовывает самостоятельно, согласно заданному стилю (правилу отрисовки). Векторные тайлы позволяют изменять внешний вид карты в реальном времени, создавать интерактивные объекты и анимации, а также обеспечивают более плавное масштабирование и перемещение по карте. Для работы с векторной картой используйте библиотеку [MapGL JS API](https://docs.2gis.com/mapgl/overview/features).

### Можно ли для работы с тайлами использовать сторонние библиотеки?[](https://docs.2gis.com/maps/overview#%D0%BC%D0%BE%D0%B6%D0%BD%D0%BE-%D0%BB%D0%B8-%D0%B4%D0%BB%D1%8F-%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%8B-%D1%81-%D1%82%D0%B0%D0%B9%D0%BB%D0%B0%D0%BC%D0%B8-%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D1%8C-%D1%81%D1%82%D0%BE%D1%80%D0%BE%D0%BD%D0%BD%D0%B8%D0%B5-%D0%B1%D0%B8%D0%B1%D0%BB%D0%B8%D0%BE%D1%82%D0%B5%D0%BA%D0%B8)

- Векторные тайлы: для доступа к Map Tiles API вы можете использовать только библиотеку MapGL JS API.
- Растровые тайлы: рекомендуется использовать библиотеку RasterJS API. Вы также можете использовать сторонние библиотеки, но их совместимость с растровыми картами 2ГИС не гарантируется и в любой момент может быть нарушена без предупреждения.

Подробнее об условиях см. в [Соглашении об использовании сервисов 2ГИС](https://law.2gis.ru/rules).

### Можно ли напрямую получить доступ к тайлам карты?[](https://docs.2gis.com/maps/overview#%D0%BC%D0%BE%D0%B6%D0%BD%D0%BE-%D0%BB%D0%B8-%D0%BD%D0%B0%D0%BF%D1%80%D1%8F%D0%BC%D1%83%D1%8E-%D0%BF%D0%BE%D0%BB%D1%83%D1%87%D0%B8%D1%82%D1%8C-%D0%B4%D0%BE%D1%81%D1%82%D1%83%D0%BF-%D0%BA-%D1%82%D0%B0%D0%B9%D0%BB%D0%B0%D0%BC-%D0%BA%D0%B0%D1%80%D1%82%D1%8B)

Чтобы получить векторные тайлы напрямую для интеграции в ГИС-систему, которая не поддерживает использование MapGL JS API, вы можете установить API-платформу 2ГИС в закрытом контуре (On-Premise) и [экспортировать векторные тайлы в формате Mapbox Vector Tiles (MVT)](https://docs.2gis.com/on-premise/api-platform/user-guide/vector-tiles). Растровые тайлы вы можете получить напрямую через Raster Tiles API и Static API.

### Какие есть технические ограничения?[](https://docs.2gis.com/maps/overview#%D0%BA%D0%B0%D0%BA%D0%B8%D0%B5-%D0%B5%D1%81%D1%82%D1%8C-%D1%82%D0%B5%D1%85%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B5-%D0%BE%D0%B3%D1%80%D0%B0%D0%BD%D0%B8%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)

MapGL JS API оптимизирован для работы в современных веб-браузерах, полный список браузеров и их версий см. в разделе [Поддержка браузеров](https://docs.2gis.com/mapgl/overview/browser-support). Если вы планируете добавлять 3D-объекты на карту, ознакомьтесь с [Требованиями и рекомендациями для 3D-моделей](https://docs.2gis.com/mapgl/map-style/immersive/models#requirements).

### Как можно настроить внешний вид карты?[](https://docs.2gis.com/maps/overview#%D0%BA%D0%B0%D0%BA-%D0%BC%D0%BE%D0%B6%D0%BD%D0%BE-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B8%D1%82%D1%8C-%D0%B2%D0%BD%D0%B5%D1%88%D0%BD%D0%B8%D0%B9-%D0%B2%D0%B8%D0%B4-%D0%BA%D0%B0%D1%80%D1%82%D1%8B)

При работе с векторными картами 2ГИС через MapGL JS API вы можете:

- Настроить центр карты, масштаб, элементы управления картой, графический режим, обработку событий и т. д.
- Изменить внешний вид карты (цвета, освещение, уровень детализации, видимость объектов и подписей и многое другое) с помощью
[Редактора стилей](https://docs.2gis.com/maps/styles/overview). - Добавить на карту свои данные: объекты разной сложности (иконки, маркеры с анимацией, 3D-модели) или источники векторных и растровых геоданных.

При работе с растровыми картами 2ГИС через Raster Tiles API и Static API вы можете:

- Настроить центр карты, масштаб, элементы управления картой, обработку событий и т. д.
- Добавить на карту свои данные: объекты (маркеры, геометрические фигуры, подписи) или геоданные в формате GeoJSON.
