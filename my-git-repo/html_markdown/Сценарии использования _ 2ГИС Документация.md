MapGL JS API совместима со следующим веб-браузерами:

## Desktop[](https://docs.2gis.com/mapgl/overview/browser-support#desktop)

- Браузеры на базе Chromium: Chrome, Microsoft Edge и другие (Windows, macOS, Linux) — две последние мажорные стабильные версии (за исключением режима IE для Microsoft Edge).
- Firefox (Windows, macOS, Linux) — две последние мажорные стабильные версии.
- Safari (macOS) — две последние мажорные стабильные версии.

## Android 7+[](https://docs.2gis.com/mapgl/overview/browser-support#android-7)

- Chrome 119+ — последняя версия с поддержкой Android 7+.
- Chrome WebView 119+ — последняя версия с поддержкой Android 7+.

## iOS 12+[](https://docs.2gis.com/mapgl/overview/browser-support#ios-12)

- Safari 12+ версии.
- WebView 12+ версии.

Присутствие браузера в списке не гарантирует корректную работу API: проблемы могут возникать на уровне аппаратного обеспечения, операционной системы или драйверов. Чтобы убедиться в поддержке API для вашей платформы, используйте [метод isSupported](https://docs.2gis.com/mapgl/reference/method#is-supported).

Чтобы использовать карты 2ГИС в неподдерживаемом браузере, рассмотрите растровую [Raster Tiles API](https://docs.2gis.com/maps/others/rastertiles/overview) или статичную [Static API](https://docs.2gis.com/maps/others/static/overview) карту или [напишите нам письмо](mailto:api@2gis.ru).
