MapGL JS API — это JavaScript-библиотека, при помощи которой вы можете добавить карту 2ГИС в ваше веб-приложение или сайт.

Библиотека позволяет:

- Добавлять трехмерную карту, использующую технологию WebGL и картографические данные 2ГИС, в веб-приложения и сайты.
[Вращать, наклонять и масштабировать](https://2gis.github.io/demo/fly/)карту при помощи мыши или сенсорных жестов.- Отображать
[планы этажей](https://docs.2gis.com/mapgl/map/configuration/floors)торговых центров и аэропортов. - Настраивать внешний вид карты с помощью
[редактора стилей](https://docs.2gis.com/maps/styles/overview), включая[3D-модели](https://docs.2gis.com/mapgl/map-style/immersive/models),[рельеф](https://docs.2gis.com/mapgl/map-style/immersive/terrain)и другие[иммерсивные возможности](https://docs.2gis.com/mapgl/map-style/immersive/overview). - Добавлять на карту
[одиночные маркеры](https://docs.2gis.com/mapgl/objects/markers)и[группы маркеров](https://docs.2gis.com/mapgl/objects/clustering). - Добавлять
[текстовые метки](https://docs.2gis.com/mapgl/objects/labels)и рисовать[геометрические фигуры](https://docs.2gis.com/mapgl/objects/geometries). - Строить
[автомобильные](https://docs.2gis.com/mapgl/map/tools/directions#car-route)и[пешеходные](https://docs.2gis.com/mapgl/map/tools/directions#pedestrian-route)маршруты. - Добавлять
[попапы](https://docs.2gis.com/mapgl/objects/popups)и[подсказки](https://docs.2gis.com/mapgl/objects/tooltips)к объектам на карте. - Настраивать
[элементы управления](https://docs.2gis.com/mapgl/map/configuration/controls)картой. - Работать с
[GeoJSON](https://docs.2gis.com/mapgl/geodata/geojson)и[растровыми источниками WMS/WMTS](https://docs.2gis.com/mapgl/geodata/raster).

Картографические данные поддерживают [стандарты OGC](https://ru.wikipedia.org/wiki/Open_Geospatial_Consortium).

Используйте только описанные в документации поля и методы MapGL JS API. Обратная совместимость для недокументированных возможностей не гарантирована.
